## 🚀 Getting Started

### 1) Clone

```bash
git clone https://github.com/rizalfirmansyah120593-byte/BEST-PORTOFOLIO-3D.git
cd BEST-PORTOFOLIO-3D
```

### 2) Install

```bash
npm install
```

### 3) Run locally

```bash
npm run dev
```

### 4) Build

```bash
npm run build
```
